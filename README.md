# World Economic Data Visualization

Exploratory data analysis and visualization of global economic indicators using Python, with a focus on geographic patterns across countries.

## Overview

This project analyzes economic data from different countries and represents it through visualizations, including choropleth maps using GeoJSON.

The goal is to identify global patterns and better understand how economic indicators vary geographically.

## Tech Stack
- Python
- Jupyter notebook
- Pandas  
- NumPy  
- Plotly

## Data

- `world_economics.csv` → economic indicators by country  
- `world.geojson` → geographical data for map visualization  

## Features

- Data cleaning and preprocessing  
- Exploratory data analysis (EDA)  
- Data visualization (charts & maps)  
- Choropleth map representation
